# MalouTest


## Start API

1. In `backend/api`  
2. install dependencies
    ```
    cd backend/api
    npm install
    ```

4. Start server
    ```
    cd backend/api
    npm start 
    ```

## Start Frontend

> NB : In package.json, replace start by start-win (on W10). 
```
cd frontend
npm install
ng serve
```
